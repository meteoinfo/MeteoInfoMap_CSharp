Set Dimensions

Only horizontal data could be operated by MIApp class at present. The ability to operate other data such as section or one dimension data will be added later. Time index and Level index could be set at present.

l         Set time index: myApp.TimeIndex = i

l         Set level index: myApp.LevelIndex = i

The default dimensions are 0. The grid or station data will be read according to the dimension setting.
