Open Data

Several open data functions are provided to open multi-source spatial or meteorological data.

l         Open map or image data file: OpenLayer(string aFile)

l         Open GrADS data file: OpenGrADSData(string aFile)

l         Open NetCDF data file: OpenNCData(string aFile)

l         Open GRIB data file: OpenGRIBData(string aFile)

l         Open MICAPS data file: OpenMICAPSData(string aFile)

l         Open ARL packed meteorological data file: OpenARLData(string aFile)

l         Open Lon/Lat station data file: OpenLonLatData(string aFile)

l         Open HYSPLIT concentration data file: OpenHYSPLITConc(string aFile)

 

Example:

myApp.OpenGrADSData(".\\Sample\\model.ctl")
