Meteorological Data

Following meteorological data were supported at present. More data format will be supported in future.

l         NetCDF grid data

l         GRIB edition 1 and 2 data

l         GrADS binary grid and station data

l         HDF EOS5 grid and swath data

l         ARL packed meteorological data

l         HYSPLIT model output data: concentration, particle and trajectory

l         METAR data (special for the cycle METAR data of NOAA, http://weather.noaa.gov/weather/metar.shtml )

l         NOAA ISH data

l         Lon/Lat Station ASCII data

l         MICAPS 1, 3, 4, 7, 11 and 13 format data

l         ESRI ASCII Grid data

l         Surfer ASCII Grid data

l         AWX data
