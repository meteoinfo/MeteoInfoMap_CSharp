生成气象数据图层

一些重载的Dislay函数用来从格点或站点数据生成气象数据图层。

l         Display(string varName)

l         Display(GridData aGridData)

l         Display(string U, string V)

l         Display(string U, string V, string varName)

l         Display(GridData UGridData, GridData VGridData)

l         Display(GridData UGridData, GridData VGridData, GridData XGridData)

l         DisplayWind(string windDir, string windSpeed)

l         DisplayTraj()

 

例如：

myApp.Display("PS")

 

UGridData = myApp.GetGridData("U")

VGridData = myApp.GetGridData("V")

myApp.Display(UGridData, VGridData)
